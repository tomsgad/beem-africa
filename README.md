This package provides a simple and crisp way to access the Beem Africa API endpoints, query data with Laravel.

## Installation

You can install the package via composer:

```bash
composer require tomsgad/beem-africa
```

## Usage

Head over to the [Documentation](https://tomsgad.github.io/beem-africa) of this package.

### Testing

```bash
composer test
```
